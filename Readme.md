# 🐉🎮 Python Mini Games Collection

A collection of fun Python games made with **Pygame**.  
Currently includes:
- **Dragon Game** (Chrome Dino-style endless runner)
- **Bouncing Ball**
- **Rainbow Snake Game**

---
